How to use

This demo showcases a WebGL2-based particle system using GPU ping-pong simulation.

  • Click/tap anywhere on the canvas to spawn particle bursts
  • Switch to CPU button toggles between GPU and CPU rendering
  • Clear Particles resets all particles
  • Particles spawn automatically from the top center
  • Watch the FPS counter to compare GPU vs CPU performance

Technical Details

  • Particle state (pos.x, pos.y, vel.x, vel.y) stored in RGBA32F texture
  • Ping-pong technique: simulate by rendering to FBO, then swap textures
  • Instanced rendering: draw all particles with a single draw call
  • texelFetch used to read per-particle position in vertex shader
  • CPU fallback uses Canvas 2D API for compatibility
